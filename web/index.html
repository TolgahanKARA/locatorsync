Çapraz Analiz

Robot Framework locator'larını Vue bileşenleriyle karşılaştırır. Kırık ve riskli locator'ları bulur.

Heal

Kırık locator'lar için otomatik düzeltme önerileri üretir ve isteğe bağlı olarak uygular.

data-test Ekle

Robot locator'larına bakarak kırılgan locatorlara sahip elementlere id ekler (Robot-Driven); Robot'ta referansı olmayan elementlere data-test ekler (Audit-Driven).

Önce "Önizle" ile neyin değişeceğini görün, satır satır seçin, sonra "Seçilileri Uygula" ile dosyalara yazın. Git kaynağı olan Vue projelerine yazma yapılamaz.

ID Ekle

Vue'da data-test olan ama id olmayan elementlere aynı değerde id ekler. Robot'taki css=[data-test='X'] locatorlarını otomatik olarak id=X ile değiştirir.

Önce "Önizle" ile neyin değişeceğini görün, satır satır seçin, sonra "Seçilileri Uygula" ile dosyalara yazın. Git kaynağı olan projelere yazma yapılamaz.
